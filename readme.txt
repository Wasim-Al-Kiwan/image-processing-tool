# Image Processing Tool

An open-source Python script for processing images. This tool allows users to copy small images and compress larger ones. It supports resizing, quality adjustment, and works with popular image formats.

## Key Features

- Copy small images to an output folder.
- Reduce the quality and resize larger images.
- Supports common image formats (JPEG, PNG).

## Technologies Used

- Python
- OpenCV

## Purpose

This project was created to provide a simple yet effective tool for image processing. Whether you need to organize your image collection or reduce file sizes, this script can help.
